Help for Termux-OS

What is it?

"Termux-OS" is the chroot system which installed over your Termux terminal. But, you can't use Termux terminal as usual after installing.

What if I have got proot system?

The system have got bootloader - GRUB. You can run or install (and run promptly) your system from it.

What about package manager? Can I install and use other utilities and repos?

Yes, you can install or remove your packages with Termux-OS's package manager which you can run from chroot menu

Can I customize OS without rewriting code?

Now - no, you can't. In the future, I'll write some plugins for customization

What about file modifying?

Termux-OS use Midnight Commander (Linux file manager). You can run it from chroot menu

What have I got after installation?

Music player, terminal locker, root locker (even if your device isn't rooted, XD), custom graphical package manager, multiboot system (you can exit from Termux-OS and run proot system)

Do I need install/uninstall anything for usage?

No, installer is install also needed packages! But the OS isn't compatible with zsh and zsh-plugins! You need uninstall zsh and zsh-plugins until installing
